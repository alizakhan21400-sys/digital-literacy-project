## Task 2: Digital Portfolio

For this task, I created my digital portfolio using three platforms: GitHub, LinkedIn, and Kaggle.

- **GitHub** is used to store and showcase my projects. I created a profile README to introduce myself and highlight my learning goals.
- **LinkedIn** is a professional networking platform where I added my educational details and started building my professional presence.
- **Kaggle** is a platform for practicing data science and programming skills, and I created an account to begin learning through challenges.

These platforms will help me develop my technical skills, build a strong online presence, and prepare for future internships and career opportunities. Over the next four years, I plan to regularly update my GitHub with projects, connect with professionals on LinkedIn, and improve my problem-solving skills on Kaggle.
